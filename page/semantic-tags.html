HTML5 시맨틱 태그 가이드

시멘틱 태그를 일상적으로 사용하는 습관을 형성하기 위해 꾸준히 연습하고 있다.

이를 위해 한눈에 알아볼 수 있는 학습 자료가 부족하여 직접 자료를 제작하였다.

문서 구조 관련 태그

<header> - 헤더

문서나 섹션의 머리말 부분을 정의합니다. 주로 제목, 로고, 네비게이션 등을 포함합니다.

웹사이트 제목

이것은 헤더 영역입니다.

<main> - 메인 콘텐츠

문서의 주요 내용을 담는 영역입니다. 문서당 하나만 사용해야 합니다.

이 부분은 문서의 메인 콘텐츠입니다.

<footer> - 푸터

문서나 섹션의 끝부분을 정의합니다. 저작권 정보, 연락처, 사이트맵 등을 포함합니다.

© 2023 웹사이트. 모든 권리 보유.

<nav> - 네비게이션

웹 사이트의 주요 탐색 링크를 모아놓은 영역을 정의합니다.

  • 홈
  • 서비스
  • 연락처

<section> - 섹션

문서의 독립적인 섹션을 정의합니다. 일반적으로 제목(h1-h6)을 포함합니다.

섹션 제목

이것은 독립적인 섹션 내용입니다.

<article> - 아티클

독립적으로 배포하거나 재사용할 수 있는 완결된 내용을 정의합니다. 뉴스 기사, 블로그 포스트, 사용자 댓글 등에 적합합니다.

기사 제목

이것은 독립적인 기사의 내용입니다.

작성자: 홍길동

<aside> - 사이드바

주요 콘텐츠와 간접적으로 연관된 부가 정보를 담는 영역을 정의합니다. 사이드바, 광고, 관련 링크 등에 적합합니다.

텍스트 강조 관련 태그

<strong> vs <b>

strong: 중요한 텍스트를 강조합니다. 의미적으로 중요성을 나타냅니다.

b: 단순히 굵은 글씨를 표시합니다. 시각적 표현만을 위한 태그입니다.

이 부분은 중요합니다. (strong 태그)

이 부분은 굵게 표시됩니다. (b 태그)

<em> vs <i>

em: 강조하는 텍스트를 정의합니다. 의미적으로 강조를 나타냅니다.

i: 단순히 이탤릭체를 표시합니다. 시각적 표현만을 위한 태그입니다.

이 부분은 강조됩니다. (em 태그)

이 부분은 이탤릭체로 표시됩니다. (i 태그)

<mark> - 하이라이트

텍스트를 하이라이트(형광펜 효과)하여 강조합니다.

이 문장에서 중요한 부분을 하이라이트했습니다.

<cite> - 인용 출처

작품의 제목(책, 노래, 영화 등)을 표시할 때 사용합니다.

토지는 박경리 작가의 대하소설입니다.

<blockquote> vs <q>

blockquote: 긴 인용문을 나타냅니다. 별도의 단락으로 표시됩니다.

q: 짧은 인용구를 나타냅니다. 인라인 요소입니다.

삶이 있는 한 희망은 있다.

- 키케로

키케로는 삶이 있는 한 희망은 있다라고 말했습니다.

특수 콘텐츠 태그

<figure> 및 <figcaption>

독립적인 콘텐츠(이미지, 다이어그램, 코드 등)와 그 설명을 그룹화합니다.

<time> - 시간

날짜나 시간을 나타냅니다. datetime 속성을 통해 기계가 읽을 수 있는 형식을 제공합니다.

공연 일시: 2023년 12월 25일 저녁 8시

<address> - 연락처 정보

연락처 정보를 나타냅니다. 주로, 이메일 주소, 전화번호, 물리적 주소 등이 포함됩니다.

웹사이트 관리자: admin@example.com
서울특별시 강남구 테헤란로 123

<details> 및 <summary>

접을 수 있는 위젯을 만듭니다. 사용자가 클릭하여 추가 정보를 볼 수 있습니다.

더 자세한 정보 보기

여기에 추가 정보가 표시됩니다. 사용자가 '더 자세한 정보 보기'를 클릭하면 이 내용이 보입니다.

폼 관련 태그

<fieldset> 및 <legend>

관련된 폼 요소들을 그룹화하고 레이블을 부여합니다.

개인 정보

<output> - 계산 결과

계산 결과를 표시하는 데 사용됩니다.

+ = 0

<progress> - 진행 상태

작업의 완료 진행 상태를 나타냅니다.

70%

<meter> - 측정값

알려진 범위 내의 스칼라 측정값을 나타냅니다. (예: 디스크 사용량, 투표율 등)

70%

HTML5에서 더 이상 지원하지 않는 태그들

<center>, <font>, <basefont>

HTML5에서는 더 이상 지원하지 않습니다. 대신 CSS를 사용하여 텍스트 스타일과 정렬을 지정해야 합니다.

<frame>, <frameset>, <noframes>

HTML5에서는 더 이상 지원하지 않습니다. 대신 iframe 또는 CSS 레이아웃을 사용해야 합니다.

<acronym>

HTML5에서는 더 이상 지원하지 않습니다. 대신 <abbr> 태그를 사용해야 합니다.

<big>, <strike>, <tt>

HTML5에서는 더 이상 지원하지 않습니다. 대신 CSS를 사용하여 텍스트 스타일을 지정해야 합니다.

<applet>

HTML5에서는 더 이상 지원하지 않습니다. 대신 <object> 태그를 사용해야 합니다.

<dir>, <menu> (이전 버전)

HTML5에서는 더 이상 이전 형태로 지원하지 않습니다. <menu>는 HTML5에서 새로운 의미로 재정의되었습니다.
